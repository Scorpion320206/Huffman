# Архиватор Хаффмана

Архиватор на основе алгоритма Хаффмана для сжатия и распаковки файлов и директорий.

## Возможности

- Сжатие одного или нескольких файлов
- Сжатие целых директорий с сохранением структуры
- Распаковка архивов с восстановлением структуры
- Поддержка бинарных файлов
- Поддержка Unicode имен файлов
- Интерактивный режим работы
- Командная строка с параметрами
